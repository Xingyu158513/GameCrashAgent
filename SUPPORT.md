# Support

Use GitHub Issues for reproducible bugs and focused feature requests after the repository is public.

Before opening an issue:

1. run the latest release;
2. keep standard redaction enabled;
3. review the generated report manually;
4. include Windows version, GameCrashAgent version, expected behavior, actual behavior, and minimal reproduction steps.

GameCrashAgent does not provide emergency repair, guaranteed root-cause identification, or automatic driver remediation.
